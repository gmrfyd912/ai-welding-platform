=============================================================
GTAW 용접 이론 및 실습 평가 - OX 퀴즈형 (7차 세트)
=============================================================
* 난이도 비율: 하(50%), 중(30%), 상(20%)
* 카테고리: 안전, 장비/도구, 용접종류, 아크이해, 결함, 비파괴검사, 재료, 극성, 전원특성, 입열량/변형

[카테고리 1: 용접과 관련된 안전]
1. (하) 산소나 가스 Regulator(조정기)를 장착할 때는 결합을 부드럽게 하기 위해 나사산에 소량의 기름(Oil)이나 구리스를 발라주는 것이 좋다. ( )
정답: X
해설: 고압의 산소나 보호가스 라인에 기름이나 구리스가 닿으면 마찰이나 압력에 의해 폭발할 위험이 있으므로 절대 금지합니다.

2. (중) 아연도금 강판을 용접할 때 발생하는 하얀색 Fume(흄)을 다량 흡입하면, 오한과 발열을 동반하는 금속열(Metal fume fever)에 걸릴 수 있다. ( )
정답: O
해설: 아연(Zinc)이 타면서 발생하는 Fume은 인체에 매우 유해하여 급성 호흡기 질환인 금속열을 유발하므로 방진마스크 착용이 필수입니다.

[카테고리 2: 장비 및 도구]
3. (하) 오염되거나 뭉툭해진 Tungsten 전극봉을 다시 뾰족하게 만들기 위해 니퍼나 펜치로 끝부분을 뚝 부러뜨려 그대로 사용해도 무방하다. ( )
정답: X
해설: 전극을 강제로 부러뜨리면 미세하게 갈라져 Arc가 흔들리므로, 반드시 전용 절단기나 그라인더로 잘라내고 세로 방향으로 연마해야 합니다.

4. (상) 식별색이 빨간색인 토륨 텅스텐(Thoriated tungsten) 전극봉은 전자 방출이 우수하지만, 토륨 성분에 미량의 방사능 물질이 포함되어 있어 연마 시 분진을 흡입하지 않도록 주의해야 한다. ( )
정답: O
해설: 2% 토륨 텅스텐은 방사성 물질 이슈가 있어 분진 흡입을 피해야 하며, 최근에는 이를 대체하기 위해 세륨(회색)이나 란탄 전극이 많이 쓰입니다.

[카테고리 3: 용접의 종류]
5. (하) GTAW 공정은 텅스텐 전극이 직접 녹아서 모재의 틈을 채워주는 소모성 전극 용접이다. ( )
정답: X
해설: GTAW의 Tungsten 전극은 열을 내기만 할 뿐 소모되지 않는(비소모성) 전극이며, 필요 시 별도의 용접봉(용가재)을 쇳물에 공급합니다.

6. (중) Laser welding(레이저 용접)은 기계적 에너지를 마찰열로 바꾸는 고상 용접(Solid-state welding)의 한 종류이다. ( )
정답: X
해설: 레이저 용접은 강력한 빛 에너지(레이저 빔)를 열원으로 사용하여 모재를 깊게 녹이는 정밀 용융 용접법입니다.

[카테고리 4: 아크의 이해]
7. (하) 용접 전류의 수치를 높이면 Arc의 열이 강해져 모재를 녹이는 힘, 즉 용입(Penetration)이 깊어진다. ( )
정답: O
해설: 전류는 입열량의 핵심 요소이므로 전류 상승은 곧 아크 열에너지의 증가와 용입 깊이 증가로 이어집니다.

8. (중) DCEN 용접 시 Tungsten 전극봉의 끝을 둥글고 뭉툭하게 연마할수록 Arc stiffness(아크 강성)가 좋아져 쇳물 제어가 쉽다. ( )
정답: X
해설: 전극 끝이 뭉툭하면 Arc가 넓게 퍼져 강성이 떨어집니다. 끝을 뾰족하게 연마해야 Arc가 뻣뻣하고 직진성 있게 집중됩니다.

[카테고리 5: 용접 결함]
9. (하) GTAW는 쇳물이 튀는 Spatter(스패터) 결함이 SMAW나 GMAW에 비해 훨씬 자주 발생하는 편이다. ( )
정답: X
해설: GTAW는 비소모성 전극을 사용하고 아크가 매우 부드러워 Spatter 발생이 전무하다시피 한 가장 깨끗한 용접법입니다.

10. (상) 용접을 마치는 부분에 쇳물이 움푹 파인 채로 굳어 생기는 Crater pipe 결함은, 급격한 수축에 의해 발생하는 균열(Crack)의 일종으로 취급된다. ( )
정답: O
해설: Crater 부위가 덜 채워지면 응고 수축 시 중심부로 응력이 집중되어 미세한 별 모양 균열(Crater crack)이 쉽게 발생합니다.

[카테고리 6: 비파괴 검사]
11. (하) 용접이 완전히 끝난 후 가장 마지막에 실시해야 하는 비파괴 검사는 VT(육안 검사)이다. ( )
정답: X
해설: VT(육안 검사)는 가장 기본적이고 선행되어야 하는 검사로, 외관상 합격해야 다음 단계(RT, UT 등)로 넘어갈 수 있습니다.

12. (중) PT(액체 침투 탐상 검사)를 실시하기 위해서는 먼저 검사할 모재에 강력한 자석을 붙여 자화시켜야 한다. ( )
정답: X
해설: 자석을 사용하는 것은 MT(자분 탐상)입니다. PT는 모세관 현상을 이용하므로 자성과는 전혀 무관합니다.

[카테고리 7: 재료 및 성분]
13. (하) 알루미늄(Al)은 철(Fe)보다 훨씬 가벼운 금속이지만, 용융점(녹는 온도)은 철보다 훨씬 높다. ( )
정답: X
해설: 순수 알루미늄의 용융점은 약 660도이며, 철(약 1530도)보다 훨씬 낮아 용접 시 순식간에 녹아내릴 수 있어 주의가 필요합니다.

14. (중) Pre-heating(예열) 작업은 용접 후 모재가 너무 빨리 식는 것을 방지하여(냉각 속도 감소), 열영향부가 딱딱해지는 것을 막아준다. ( )
정답: O
해설: 예열은 모재 전체의 온도를 올려 급랭을 막음으로써 조직의 경화와 균열을 예방하는 핵심 기술입니다.

[카테고리 8: 용접 극성]
15. (하) 탄소강(Carbon steel) 배관을 GTAW로 용접할 때는 기본적으로 AC(교류) 전원을 설정하여 작업한다. ( )
정답: X
해설: 탄소강이나 스테인리스강은 깊은 용입을 얻기 위해 DCEN(직류 정극성)을 사용하는 것이 표준입니다.

16. (중) 교류(AC) 알루미늄 용접에서 전극이 (+), 모재가 (-)가 되는 반 사이클(Half-cycle) 동안 표면의 산화막이 깨지는 Cleaning action이 일어난다. ( )
정답: O
해설: 모재(-)에서 전극(+)으로 튀어 오르는 전자들이 모재 표면의 단단한 산화알루미늄 피막을 타격하여 부수는 원리입니다.

[카테고리 9: 전원 특성]
17. (하) 용접기의 무부하 전압(Open-circuit voltage)은 용접 중 아크가 켜져 있을 때 흐르는 아크 전압보다 항상 낮다. ( )
정답: X
해설: 무부하 전압(기계가 켜져만 있는 상태)은 아크를 최초로 발생시키기 위해 약 60~80V로 높게 설정되어 있으며, 용접이 시작되면 아크 전압(10~20V)으로 뚝 떨어집니다.

18. (상) 수동 용접기(정전류 특성)는 작업자의 손떨림으로 Arc length가 살짝 길어지면, 전류가 수십 암페어(A)씩 크게 출렁여 쇳물 제어를 어렵게 한다. ( )
정답: X
해설: 정전류 특성은 Arc length가 변하여 전압이 오르락내리락해도 전류의 변화폭이 극히 적어(일정하게 유지) 쇳물 제어를 오히려 편하게 해줍니다.

[카테고리 10: 입열량과 변형]
19. (하) 용접 시 Weaving(위빙) 폭을 너무 넓게 하여 운봉 속도가 느려지면 모재에 가해지는 열이 많아져 변형이 커진다. ( )
정답: O
해설: Weaving이 과도하면 속도가 느려지고 입열량(H)이 증가하여 금속의 팽창과 수축이 심해집니다.

20. (상) 두꺼운 판의 초층 용접(Root pass) 직후 잔류 응력을 제거하기 위해, 얇은 초층 비드 위에 즉각적으로 Peening(피닝) 타격을 가하는 것이 권장된다. ( )
정답: X
해설: 초층 비드는 두께가 얇고 취약하여 Peening(해머질)을 가하면 그 충격으로 즉시 크랙(균열)이 발생할 수 있으므로 절대 피닝을 하지 않습니다.

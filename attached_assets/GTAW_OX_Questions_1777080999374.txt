=============================================================
GTAW(TIG) 용접 이론 및 실습 평가 - OX 퀴즈형 (1차 세트)
=============================================================
* 난이도 비율: 하(50%), 중(30%), 상(20%)
* 카테고리: 안전, 장비/도구, 용접종류, 아크이해, 결함, 비파괴검사, 재료, 극성, 전원특성, 입열량/변형

[카테고리 1: 용접과 관련된 안전]
1. (하) 용접 후 뜨거운 모재를 다룰 때는 얇은 면장갑 위에 젖은 장갑을 껴서 열을 식혀야 한다. ( )
정답: X
해설: 젖은 장갑은 열전도를 높여 화상을 입기 쉬우며 감전의 위험이 있으므로 반드시 마른 가죽장갑을 착용해야 합니다.

2. (중) 아르곤(Ar) 가스는 인체에 직접적인 독성은 없으나, 밀폐공간에서는 산소 결핍을 유발할 수 있는 질식성 가스이다. ( )
정답: O
해설: 아르곤은 불활성 가스 독성은 없지만, 공기보다 무거워 바닥에 쌓이면 산소를 밀어내어 질식을 유발합니다.

[카테고리 2: 장비 및 도구]
3. (하) 가스 렌즈(Gas lens)를 사용하면 쉴드 가스의 난류를 방지하고 직진성을 높여주어 텅스텐 전극을 더 길게 뽑아서 용접할 수 있다. ( )
정답: O
해설: 가스 렌즈 내부의 미세한 망이 가스를 층류로 만들어 가스 보호 효과를 극대화해줍니다.

4. (상) 용접기 패널의 '초기 가스(Pre-flow)' 설정 시간은 용접 모재의 두께가 두꺼울수록 짧게 설정해야 한다. ( )
정답: X
해설: 초기 가스는 아크 발생 전 배관 및 토치 내부의 공기를 밀어내는 역할이므로 모재 두께보다는 토치 길이와 관계가 깊습니다.

[카테고리 3: 용접의 종류]
5. (하) GTAW 용접은 전극 자체가 녹아 용융지로 들어가는 소모성 전극식 용접법이다. ( )
정답: X
해설: GTAW(TIG)의 텅스텐 전극은 열에 강해 녹지 않는 비소모성 전극이며, 필요시 별도의 용가재(용접봉)를 공급합니다.

[카테고리 4: 아크의 이해]
6. (중) 용접 시 전류를 일정하게 두었을 때, 아크 길이를 길게 하면 아크 전압은 낮아진다. ( )
정답: X
해설: 아크 길이가 길어지면 전기적 저항이 증가하므로 이를 극복하기 위해 아크 전압은 상승합니다.

[카테고리 5: 용접 결함]
7. (하) 언더컷(Undercut)은 모재 가장자리가 파인 상태로 용착금속이 채워지지 않은 결함으로, 주로 전류가 너무 높거나 운봉이 빠를 때 발생한다. ( )
정답: O
해설: 과도한 열로 모재가 녹아 파였으나 용가재가 충분히 채워지지 않아 발생하는 대표적 결함입니다.

8. (중) 크레이터(Crater) 부위에 발생하는 균열을 방지하기 위해서는 용접 종료 시 전류를 서서히 낮추는 다운슬로프 기능을 꺼두는 것이 좋다. ( )
정답: X
해설: 크레이터 균열 방지를 위해서는 반드시 다운슬로프(전류 점감기능)를 활성화하여 용융지가 서서히 굳도록 해야 합니다.

[카테고리 6: 비파괴 검사]
9. (상) 자분탐상검사(MT)는 스테인리스강(오스테나이트계)이나 알루미늄 용접부의 표면 결함을 검출하는 데 가장 효과적인 방법이다. ( )
정답: X
해설: 자분탐상검사(MT)는 자성을 띠는 철강 재료에만 적용 가능합니다. 알루미늄이나 오스테나이트계 스테인리스는 비자성이므로 침투탐상(PT)을 해야 합니다.

[카테고리 7: 재료 및 성분]
10. (중) 탄소강은 탄소(C) 함유량이 높아질수록 경도와 강도가 증가하지만, 용접성은 오히려 떨어진다. ( )
정답: O
해설: 탄소량이 많아지면 용접 후 냉각 시 균열(Crack)이 발생하기 쉬워 예열 등의 추가 조치가 필요합니다.

[카테고리 8: 용접 극성]
11. (하) 직류 정극성(DCEN)은 모재가 (+), 텅스텐 전극이 (-)인 상태를 말하며 모재에 깊은 용입을 얻을 수 있다. ( )
정답: O
해설: 전자는 (-)에서 (+)로 이동하며 충돌하므로, 모재가 (+)일 때 모재 쪽에 열이 약 70% 집중되어 용입이 깊어집니다.

[카테고리 9: 전원 특성]
12. (상) 자동 용접인 GMAW(MIG/MAG)에는 주로 정전류 특성 전원과 가변송급속도 방식이 결합되어 사용된다. ( )
정답: X
해설: GMAW(반자동/자동용접)에는 자기제어 특성이 있는 '정전압 특성' 전원을 주로 사용합니다.

[카테고리 10: 입열량과 변형]
13. (중) 용접 변형을 최소화하려면 용접 속도를 느리게 하여 용접부에 입열량을 최대한 많이 가해야 한다. ( )
정답: X
해설: 입열량이 많을수록 열팽창과 수축이 커져 변형이 심해집니다. 변형을 줄이려면 속도를 높여 입열량을 줄여야 합니다.

=============================================================
GTAW 용접 이론 및 실습 평가 - OX 퀴즈형 (10차 세트 - 최종)
=============================================================
* 난이도 비율: 하(50%), 중(30%), 상(20%)
* 카테고리: 안전, 장비/도구, 용접종류, 아크이해, 결함, 비파괴검사, 재료, 극성, 전원특성, 입열량/변형

[카테고리 1: 용접과 관련된 안전]
1. (하) 밀폐된 공간에서 Ar gas가 누출되면 공기보다 무거워 바닥으로 가라앉으므로, 하단 배기를 통한 환기가 필수적이다. ( )
정답: O
해설: Ar은 질식성 가스이므로 밀폐공간 바닥에 고이는 것을 막기 위해 하단 강제 배기를 실시해야 합니다.

2. (중) High frequency(고주파) Arc start 장치가 켜져 있으면, Tungsten 전극이 모재에 직접 닿지 않아도 공중에서 스파크가 튀며 Arc가 발생한다. ( )
정답: O
해설: 고주파 전압이 공기 절연을 파괴하므로 비접촉 기동이 가능하여 Tungsten 오염을 방지합니다.

[카테고리 2: 장비 및 도구]
3. (하) Nozzle(세라믹 컵)의 크기를 나타내는 번호(예: #6, #8)는 숫자가 클수록 Nozzle의 내경이 작다는 것을 의미한다. ( )
정답: X
해설: Nozzle 번호는 1/16 inch 단위이므로, 숫자가 클수록 내경(가스 방출구)도 커집니다.

4. (중) Gas lens 부품을 사용하면 보호 가스가 흩어지는 난류(Turbulence) 상태를 일직선으로 뻗는 층류(Laminar flow)로 바꾸어 준다. ( )
정답: O
해설: Gas lens 내부의 미세 망이 가스를 곧게 뻗게 만들어 가스 보호 능력을 극대화합니다.

[카테고리 3: 용접의 종류]
5. (하) FCAW는 튜브형 Wire 내부에 Flux가 들어있는 소모성 전극을 사용하는 반자동 용접 공정이다. ( )
정답: O
해설: 플럭스가 채워진 와이어가 연속 공급되며 슬래그가 쇳물을 보호하는 널리 쓰이는 공정입니다.

6. (중) SAW(Submerged Arc Welding)는 얇은 박판을 세밀하게 용접할 때 주로 사용하는 수동 용접기법이다. ( )
정답: X
해설: SAW(잠호 용접)는 두꺼운 후판 강판을 하향으로 자동 용접할 때 쓰이는 대형 자동화 장비입니다.

[카테고리 4: 아크의 이해]
7. (하) Arc length(아크 길이)를 짧게 유지하면 Arc Voltage(전압)도 낮아지며 비드 폭이 좁아진다. ( )
정답: O
해설: Arc 길이는 전압에 비례하며, 짧을수록 열이 집중되어 폭이 좁고 용입이 깊어집니다.

8. (상) Arc stiffness(아크 강성)가 좋다는 것은 Arc가 부드럽게 퍼지지 않고 직진성이 강해 열을 한 곳에 정확히 모을 수 있음을 의미한다. ( )
정답: O
해설: Tungsten 전극을 뾰족하게 연마하면 이 Arc 강성이 좋아져 정밀한 제어가 가능합니다.

[카테고리 5: 용접 결함]
9. (하) Undercut 결함은 Current가 너무 낮고 속도가 느려 쇳물이 단순히 모재 위에 얹혀진 상태를 말한다. ( )
정답: X
해설: 설명은 Overlap 결함에 해당합니다. Undercut은 너무 높은 Current로 인해 모재가 파인 결함입니다.

10. (중) 용융지(Pool) 끝부분의 전류를 서서히 줄이지 않고 확 꺼버리면, 급격한 수축에 의해 파이프 모양으로 홈이 남는 Crater pipe 결함이 생긴다. ( )
정답: O
해설: 이를 막기 위해 용접기 패널의 Down-slope 기능을 반드시 활성화해야 합니다.

11. (상) Tungsten inclusion 결함은 RT(방사선 투과 검사) 필름을 판독할 때 주변보다 검고 어두운 색으로 나타난다. ( )
정답: X
해설: Tungsten은 밀도가 매우 높아 방사선을 흡수하므로, 필름 상에서 주변보다 밝은(하얀) 점으로 나타납니다.

[카테고리 6: 비파괴 검사]
12. (하) PT(침투 탐상 검사)는 자석에 붙지 않는 Aluminum이나 Austenitic STS의 표면 결함을 검출하는 데 매우 효과적이다. ( )
정답: O
해설: 자성이 필요 없는 모세관 현상을 이용하므로 거의 모든 비다공성 금속의 표면 검사에 쓰입니다.

13. (중) MT(자분 탐상 검사)는 모재 내부 깊숙한 중심부에 있는 결함의 위치와 깊이를 정확하게 알 수 있다. ( )
정답: X
해설: MT는 표면 결함이나 표면 바로 아래(2~3mm 이내) 결함만 검출 가능하며, 깊은 내부는 UT나 RT를 써야 합니다.

[카테고리 7: 재료 및 성분]
14. (하) Aluminum의 용융점은 약 660℃로 철(Fe)에 비해 매우 낮지만, 표면을 덮고 있는 산화막의 용융점은 약 2050℃로 매우 높다. ( )
정답: O
해설: 융점 차이가 커서 산화막을 Cleaning action(청정 작용)으로 먼저 부수지 않으면 용접이 불가능합니다.

15. (상) Stainless pipe를 GTAW로 용접할 때, 내부에 Ar gas를 채워넣는 Back purging을 실시하는 이유는 Sugaring(이면 산화)을 방지하기 위함이다. ( )
정답: O
해설: 산소와 결합하여 까맣게 타고 부식성이 취약해지는 것을 막기 위해 내부 공기를 차단하는 필수 작업입니다.

[카테고리 8: 용접 극성]
16. (하) DCEN(직류 정극성)은 전자가 모재에 충돌하므로 모재에 열이 집중되어 깊은 Penetration(용입)을 만든다. ( )
정답: O
해설: GTAW 철강 용접에서 가장 표준적으로 사용되는 극성입니다.

17. (상) Aluminum 교류 용접에서 Cleaning action(청정 작용)은 모재가 (-)이고 전극이 (+)인 EP(역극성) 구간에서 전자가 튀어나가며 산화막을 부수는 현상이다. ( )
정답: O
해설: 교류 1사이클 내에서 이 EP 구간이 산화막을 파괴하고, EN 구간이 모재를 녹입니다.

[카테고리 9: 전원 특성]
18. (중) Constant Voltage(정전압 특성) 전원은 주로 수동 GTAW 용접에 쓰이며, 손떨림에도 일정한 전류를 유지해준다. ( )
정답: X
해설: 수동 용접에 쓰이는 것은 Constant Current(정전류 특성)입니다. 정전압 특성은 GMAW(반자동)에 쓰입니다.

19. (상) 용접기의 OCV(Open Circuit Voltage)가 높으면 Arc Start는 쉬워지지만 감전(Electric shock)의 위험성은 커진다. ( )
정답: O
해설: 무부하 전압이 높으면 절연을 뚫기 쉬워 아크 발생엔 유리하지만 인체엔 치명적일 수 있습니다.

[카테고리 10: 입열량과 변형]
20. (중) 용접 구간을 잘게 나누고 전체 용접 진행 방향과 반대로 용접해 나가는 Back-step 기법은 Residual stress와 Deformation을 억제하는 데 탁월하다. ( )
정답: O
해설: 열 누적을 분산시키고 잔류 응력을 상쇄하여 변형을 방지하는 실무적인 운봉 기법입니다.

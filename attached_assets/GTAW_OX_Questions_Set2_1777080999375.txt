=============================================================
GTAW(TIG) 용접 이론 및 실습 평가 - OX 퀴즈형 (2차 세트)
=============================================================
* 난이도 비율: 하(50%), 중(30%), 상(20%)
* 카테고리: 안전, 장비/도구, 용접종류, 아크이해, 결함, 비파괴검사, 재료, 극성, 전원특성, 입열량/변형

[카테고리 1: 용접과 관련된 안전]
1. (하) 용접기의 접지(어스)선은 아크 쏠림이나 저항을 줄이기 위해 용접하는 부위와 가급적 가까운 곳에 튼튼하게 연결하는 것이 좋다. ( )
정답: O
해설: 접지점(어스)이 용접부에서 너무 멀거나 부실하면 전류 흐름이 불안정해지고 아크 쏠림(Arc blow)이 발생하기 쉽습니다.

[카테고리 2: 장비 및 도구]
2. (하) 알루미늄 용접 시 교류(AC)를 사용할 때는 텅스텐 전극봉의 끝을 바늘처럼 뾰족하게 연마하여 사용해야 한다. ( )
정답: X
해설: 직류(DC) 용접 시에는 끝을 뾰족하게 하지만, 알루미늄(AC) 용접 시에는 전극에 열이 많이 발생하므로 끝을 약간 둥글게(Ball 형) 만들어 아크를 안정시킵니다.

3. (상) 가스 렌즈(Gas lens)를 사용하면 가스 유속이 빨라져 바람이 부는 야외에서도 완벽하게 가스 쉴드가 가능하다. ( )
정답: X
해설: 가스 렌즈는 층류를 만들어 쉴드 범위를 넓혀주지만, 바람이 부는 곳에서는 TIG 용접 자체가 불가능하므로 방풍막(텐트 등)을 설치해야 합니다.

[카테고리 3: 용접의 종류]
4. (하) 마찰압접(Friction welding)은 고상용접의 일종으로, 금속을 녹이지 않고 마찰열로 달군 후 강한 압력으로 결합한다. ( )
정답: O
해설: 마찰용접은 외부의 열원이 아닌 부품 간의 물리적 회전 마찰열을 이용한 대표적인 압접(고상용접)입니다.

5. (중) 브레이징(경납땜)은 용접 후 변형이 거의 없고 서로 다른 이종 금속을 접합하기 쉬운 장점이 있다. ( )
정답: O
해설: 모재를 녹이지 않는 비교적 낮은 온도에서 작업하므로 열변형이 적고, 철과 구리 등 성질이 다른 금속도 접합이 가능합니다.

[카테고리 4: 아크의 이해]
6. (하) 용접 중 아크 길이를 짧게 유지하면 아크 전압이 낮아지고 용접 비드의 폭이 좁아진다. ( )
정답: O
해설: 아크 길이에 비례하여 아크 전압이 변하며, 길이가 짧으면 열 분산이 적어져 비드 폭이 좁고 집중된 용입을 얻습니다.

7. (중) 아크 쏠림(Magnetic Arc Blow) 현상은 직류(DC)보다 교류(AC) 용접을 할 때 훨씬 심하게 발생한다. ( )
정답: X
해설: 교류는 전류의 방향이 계속 바뀌어 자장이 상쇄되므로 아크 쏠림이 거의 일어나지 않으며, 주로 직류(DC)에서 발생합니다.

[카테고리 5: 용접 결함]
8. (하) 크레이터 파이프(Crater pipe)는 용접이 끝나는 지점의 쇳물이 급격히 수축하여 움푹 파이고 내부에 빈 공간이 생기는 결함이다. ( )
정답: O
해설: 크레이터 처리를 하지 않고 아크를 갑자기 끄면 발생하는 대표적인 수축 결함입니다.

9. (중) 개재물 혼입(Slag inclusion)은 GTAW(TIG) 용접에서 피복아크용접(SMAW)보다 훨씬 더 자주 발생하는 결함이다. ( )
정답: X
해설: GTAW는 플럭스(피복제)를 사용하지 않아 슬래그가 발생하지 않으므로, 슬래그 혼입 결함은 거의 발생하지 않습니다(텅스텐 혼입은 발생 가능).

10. (상) 용접 중 전극봉이 용융지나 용가재에 닿아 부러진 채로 비드 속에 남게 되는 결함을 '텅스텐 혼입'이라고 하며, 이는 주로 RT(방사선) 검사 시 하얗게(밝게) 나타난다. ( )
정답: O
해설: 텅스텐은 밀도(비중)가 철보다 훨씬 높아 방사선을 잘 흡수하므로 필름 상에 기공(검은색)과 반대로 밝은 흰색 점으로 나타납니다.

[카테고리 6: 비파괴 검사]
11. (하) 침투탐상검사(PT)는 모재의 재질이 철이든 비철이든 상관없이 표면에 열려 있는 결함을 찾는 데 사용할 수 있다. ( )
정답: O
해설: PT는 자성과 무관하므로 플라스틱이나 세라믹을 제외한 거의 모든 금속(알루미늄, 스텐 등) 표면 검사에 적용 가능합니다.

12. (중) 액체침투탐상(PT) 검사 순서는 '전처리 -> 세척 -> 침투 -> 현상 -> 관찰' 순으로 진행된다. ( )
정답: X
해설: 올바른 순서는 '전처리(세척) -> 침투액 적용 -> 세척(남은 침투액 제거) -> 현상액 적용 -> 관찰' 입니다.

[카테고리 7: 재료 및 성분]
13. (중) 탄소강 용접 시 모재를 미리 가열하는 예열(Pre-heating)은 용접 후 냉각 속도를 늦춰주어 저온 균열을 방지하는 효과가 있다. ( )
정답: O
해설: 예열은 모재와 용접부 간의 온도차를 줄여 급랭을 막고, 수소를 배출시켜 균열을 예방합니다.

[카테고리 8: 용접 극성]
14. (하) 알루미늄 용접에 사용되는 고주파 교류(AC) 전원은 정극성(DCEN)과 역극성(DCEP)의 특징이 번갈아 나타난다. ( )
정답: O
해설: 교류 1사이클 내에서 정극성(+)으로 용입을 얻고, 역극성(-)으로 산화막을 파괴하는 청정작용을 반복합니다.

15. (중) 고주파(High Frequency) 발생 장치가 켜져 있으면, 텅스텐 전극이 모재에 닿지 않아도 공중에서 스파크가 튀어 아크가 쉽게 발생한다. ( )
정답: O
해설: 고주파 전압이 공기 절연을 파괴하여 비접촉 상태에서도 아크 기동(Arc start)을 가능하게 하며, 텅스텐 오염을 방지합니다.

[카테고리 9: 전원 특성]
16. (하) 정전압 특성 용접기는 작업자가 손으로 아크 길이를 길게 하거나 짧게 해도 용접 전류가 일정하게 유지되는 전원이다. ( )
정답: X
해설: 손떨림에 의해 아크 길이에 상관없이 전류가 일정하게 유지되는 것은 수하특성(정전류 특성)이며, TIG 용접기에 사용됩니다.

[카테고리 10: 입열량과 변형]
17. (중) 용접 속도가 동일할 때, 용접 전류를 100A에서 150A로 올리면 입열량은 증가한다. ( )
정답: O
해설: 입열량(H=60EI/V)은 전압(E)과 전류(I)에 비례하므로 전류가 커지면 입열량도 커집니다.

18. (중) 용접 비드를 한쪽 끝에서부터 연속적으로 끝까지 용접(전진법)하는 것이 부분적으로 건너뛰며 용접하는 스킵(Skip)법보다 변형이 적다. ( )
정답: X
해설: 한 방향으로 계속 열을 가하면 변형이 심해지므로, 대칭법, 후퇴법, 스킵법 등을 사용하여 열을 분산시켜야 변형이 적습니다.

19. (상) 용접 중 구속(Restraint)이 강한 상태, 즉 모재가 움직이지 못하도록 단단히 고정된 상태로 용접하면 변형은 줄어들지만 잔류 응력은 커져 균열의 위험이 높아진다. ( )
정답: O
해설: 강제로 움직이지 못하게 막으면 외형 변형은 적지만, 금속 내부에서 수축하려는 힘(잔류 응력)이 그대로 갇혀서 크랙(균열)으로 이어지기 쉽습니다.

20. (상) 후열처리(Post-Weld Heat Treatment) 시 가열 온도를 너무 높여 모재의 변태점 이상으로 올리면 금속의 기계적 성질이 크게 약화될 수 있다. ( )
정답: O
해설: 잔류응력 제거를 위한 후열처리는 반드시 모재의 재결정 온도(변태점) 이하에서 실시해야 원래 금속의 강도와 성질을 유지할 수 있습니다.

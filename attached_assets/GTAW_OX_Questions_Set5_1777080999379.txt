=============================================================
GTAW 용접 이론 및 실습 평가 - OX 퀴즈형 (5차 세트)
=============================================================
* 난이도 비율: 하(50%), 중(30%), 상(20%)
* 카테고리: 안전, 장비/도구, 용접종류, 아크이해, 결함, 비파괴검사, 재료, 극성, 전원특성, 입열량/변형

[카테고리 1: 용접과 관련된 안전]
1. (하) 고압 가스 실린더(Cylinder)의 밸브를 열 때는 밸브 정면에서 얼굴을 마주 보고 신속하게 확 열어야 이물질이 잘 빠진다. ( )
정답: X
해설: 고압 가스 방출 시 압력으로 인해 파편이 튈 수 있으므로 반드시 밸브 측면(옆)에 서서 천천히 열어야 합니다.

2. (중) 화재를 A, B, C급으로 분류할 때, C급 화재는 전기로 인한 화재를 의미하며 이때는 물을 뿌리면 감전 위험이 있다. ( )
정답: O
해설: C급(전기 화재)은 CO2(이산화탄소)나 분말 소화기 등 전기가 통하지 않는 소화 약제를 사용해야 합니다.

[카테고리 2: 장비 및 도구]
3. (하) Torch 내부의 Collet(콜릿)은 Tungsten 전극의 굵기가 1.6mm든 2.4mm든 상관없이 하나의 사이즈로 모두 꽉 조여 사용할 수 있다. ( )
정답: X
해설: Collet은 텅스텐 전극의 직경(굵기)과 정확히 일치하는 사이즈를 사용해야 전기가 제대로 통하고 전극이 흔들리지 않습니다.

4. (하) Gas lens 부품을 Torch에 장착하면 Shielding gas의 층류가 형성되어 도달 거리가 길어지므로, Tungsten 전극을 Nozzle 밖으로 조금 더 길게 뽑아서 용접할 수 있다. ( )
정답: O
해설: Gas lens는 난류를 억제하고 보호가스를 일직선으로 모아주어 깊은 홈 용접 시 전극을 길게 뺄 수 있게 해줍니다.

5. (상) Pulse GTAW 기능 사용 시, 전류가 높은 Peak time(피크 시간)의 유지 비율(%)을 낮추면 모재에 가해지는 전체 입열량(Heat input)은 증가한다. ( )
정답: X
해설: Peak time은 모재를 녹이는 고전류 구간이므로, 이 시간을 낮추면(짧게 하면) 전체 열에너지가 줄어들어 입열량이 감소합니다.

[카테고리 3: 용접의 종류]
6. (하) FCAW (Flux Cored Arc Welding)는 내부가 비어있고 그 안에 Flux(용제)가 채워져 있는 와이어를 사용하는 소모성 전극 용접법이다. ( )
정답: O
해설: 플럭스 코어드 와이어를 연속으로 공급하며 용접하는 널리 쓰이는 반자동 용접입니다.

7. (중) Electroslag welding(일렉트로슬래그 용접)은 두꺼운 철판(후판)의 수직 상진 용접에 주로 사용되는 고상 용접(Solid-state welding)이다. ( )
정답: X
해설: 일렉트로슬래그 용접은 슬래그의 저항열로 모재와 용접봉을 녹이는 '용융 용접'에 속합니다.

[카테고리 4: 아크의 이해]
8. (하) 용접 중 Arc 전압은 Arc length(길이)에 비례하므로, Torch를 모재에서 높이 띄울수록 전압계의 수치는 올라간다. ( )
정답: O
해설: Arc 길이가 길어지면 전기 저항이 커져 전압이 동반 상승하는 것이 아크의 기본 특성입니다.

9. (상) GTAW 용접 시 Ar(아르곤) 가스에 He(헬륨)을 혼합하여 사용하면, 순수 Ar을 사용할 때보다 Arc 전압이 낮아지고 용입이 얕아진다. ( )
정답: X
해설: 헬륨(He)은 이온화 전압이 아르곤보다 높아 He을 섞으면 Arc 전압이 크게 상승하고 모재에 들어가는 열에너지가 커져 용입이 아주 깊어집니다.

[카테고리 5: 용접 결함]
10. (하) Spatter(스패터)는 용접 중 쇳물이 사방으로 튀어 모재에 들러붙는 현상으로, GTAW 공정은 SMAW에 비해 Spatter 발생이 극히 드물다. ( )
정답: O
해설: GTAW는 비소모성 전극과 보호가스를 사용하여 Arc가 매우 정숙하므로 Spatter가 거의 없습니다.

11. (중) Undercut 결함은 용접 전류가 너무 낮고 운봉 속도가 느릴 때 모재가 녹지 못해서 주로 발생한다. ( )
정답: X
해설: Undercut은 전류가 너무 '높거나' 운봉 속도가 너무 '빠를 때' 과도한 열로 모재가 파인 후 쇳물이 채워지지 않아 발생합니다.

12. (상) Hot crack(고온 균열)은 용접 쇳물이 굳기 시작하는 고온 단계에서 결정 입계에 모인 유황(S)이나 인(P) 같은 불순물로 인해 발생하기 쉽다. ( )
정답: O
해설: S, P 등 융점이 낮은 불순물이 금속 응고 시 마지막까지 남아 결합력을 약하게 만들어 갈라지게 합니다.

[카테고리 6: 비파괴 검사]
13. (하) 검사원의 눈과 손전등, 돋보기 등을 이용하는 육안 검사(VT)는 용접부 내부 깊숙이 위치한 미세한 Porosity를 찾아내는 데 매우 효과적이다. ( )
정답: X
해설: VT(육안 검사)는 겉으로 드러난 표면 결함(비드 외관, Undercut 등)만 확인할 수 있습니다.

14. (중) UT (초음파 탐상 검사)는 내부 결함을 찾는 데 뛰어나며, RT(방사선 투과 검사)와 달리 결함이 표면으로부터 얼마나 깊은 곳에 있는지(깊이)를 정확히 알 수 있는 장점이 있다. ( )
정답: O
해설: 초음파가 부딪혀 돌아오는 시간차를 계산하므로 결함의 수직적 깊이(Depth) 파악에 탁월합니다.

[카테고리 7: 재료 및 성분]
15. (하) 알루미늄(Al) 금속은 비중이 약 2.7로, 철(Fe, 비중 약 7.8)에 비해 매우 가벼운 경금속에 속한다. ( )
정답: O
해설: 알루미늄은 가볍고 열전도율이 높은 대표적인 비철 금속입니다.

16. (중) 황동(Brass)은 구리(Cu)와 주석(Sn)을 주성분으로 하는 합금을 일컫는 말이다. ( )
정답: X
해설: 황동(놋쇠, Brass)은 구리와 아연(Zn)의 합금이며, 구리와 주석(Sn)의 합금은 청동(Bronze)입니다.

[카테고리 8: 용접 극성]
17. (하) AC(교류) 용접 시에는 전압의 +와 - 극성이 주기적으로 계속 바뀌므로, 용접 중 DCEN(용입 깊음)과 DCEP(산화막 파괴)의 특성이 번갈아 나타난다. ( )
정답: O
해설: 1초에 수십 번씩 극성이 교차하므로 알루미늄 용접 시 용입과 청정 작용을 동시에 얻을 수 있습니다.

[카테고리 9: 전원 특성]
18. (중) 정전압 특성을 가진 반자동 용접기(GMAW 등)에서는 Torch가 멀어져 Arc length가 길어지면, 기계가 스스로 전류를 급격히 줄여 와이어 녹는 속도를 늦춤으로써 원래 Arc length를 회복한다. ( )
정답: O
해설: 이를 '자기 제어 특성(Self-regulating characteristic)'이라 하며 정전압 전원의 핵심 원리입니다.

[카테고리 10: 입열량과 변형]
19. (하) 용접 공식(H=60EI/V)에 따라, 용접 속도(V)를 느리게 진행할수록 모재 단위 길이당 들어가는 입열량(Heat input)은 더 작아진다. ( )
정답: X
해설: 운봉 속도가 느려지면 같은 구간에 열이 더 오래 가해지므로 입열량은 '커집니다(증가)'.

20. (상) Pre-heating(예열)을 실시하면 모재와 용접부 간의 온도 차이를 줄여 냉각 속도가 느려지므로, 열영향부(HAZ)의 경화를 막고 수소 지연 균열(저온 균열)을 예방할 수 있다. ( )
정답: O
해설: 예열은 금속을 천천히 굳게 하여 유해 가스(수소)가 빠져나갈 시간을 벌어주고 단단하게 깨지는 성질을 줄여줍니다.

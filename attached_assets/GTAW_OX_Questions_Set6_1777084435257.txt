=============================================================
GTAW 용접 이론 및 실습 평가 - OX 퀴즈형 (6차 세트)
=============================================================
* 난이도 비율: 하(50%), 중(30%), 상(20%)
* 카테고리: 안전, 장비/도구, 용접종류, 아크이해, 결함, 비파괴검사, 재료, 극성, 전원특성, 입열량/변형

[카테고리 1: 용접과 관련된 안전]
1. (하) 용접면의 차광 유리(Filter plate)는 용접 전류가 높을수록 차광 번호가 높은(더 어두운) 것을 사용해야 한다. ( )
정답: O
해설: 용접 전류가 높을수록 발생하는 자외선과 가시광선이 강해지므로, 눈을 보호하기 위해 더 어두운 번호의 유리를 장착해야 합니다.

2. (하) 밀폐 공간에 고압 가스 용기(Cylinder)를 보관하는 것은 환기가 안 되더라도 작업의 편리성을 위해 허용된다. ( )
정답: X
해설: 밀폐 공간에 가스 용기를 두면 미세한 누출로도 산소 결핍(질식)이나 폭발이 일어날 수 있으므로 절대 반입이나 보관을 해서는 안 됩니다.

[카테고리 2: 장비 및 도구]
3. (하) Tungsten 전극봉을 연마할 때는 연마석의 결이 전극봉의 가로 방향이 되도록 갈아야 Arc가 똑바로 모아진다. ( )
정답: X
해설: 가로 방향으로 연마하면 Arc가 흔들리고 흩어지게 됩니다. 반드시 전극봉의 길이 방향(세로 방향)으로 결이 나도록 연마해야 합니다.

4. (상) Gas lens를 사용하면 Shielding gas가 난류(Turbulence) 상태가 되어 주변 공기를 밀어내는 힘이 강해진다. ( )
정답: X
해설: Gas lens는 가스를 소용돌이치는 난류가 아니라, 일직선으로 곱게 뻗는 층류(Laminar flow) 상태로 만들어 주어 보호막을 극대화하는 부품입니다.

[카테고리 3: 용접의 종류]
5. (하) Brazing(경납땜)은 모재를 융점 이상으로 녹이지 않고, 용가재만 녹여서 틈새로 스며들게 하는 모세관 현상을 주로 이용한다. ( )
정답: O
해설: 모재를 녹이지 않으므로 열변형이 적고, 좁은 간극 사이로 용융된 용가재가 빨려 들어가는 힘(모세관 현상)으로 접합합니다.

[카테고리 4: 아크의 이해]
6. (중) Arc의 온도는 주변부보다 중심부가 온도가 더 높으며, 이 가장 뜨거운 중심부를 Arc core(아크 코어)라고 부른다. ( )
정답: O
해설: Arc core는 플라즈마 상태의 중심 기둥으로 열 집중도가 가장 높은 영역입니다.

7. (하) 용접 전류의 수치를 낮추면 Arc 열이 약해져서 모재가 깊게 녹지 않는다. ( )
정답: O
해설: 전류는 입열량과 직결되므로, 전류가 낮으면 모재를 녹이는 힘(용입 깊이)이 얕아집니다.

[카테고리 5: 용접 결함]
8. (하) Undercut은 용접 전류가 너무 낮고 용접 속도가 느릴 때 모재가 녹지 않아 생기는 결함이다. ( )
정답: X
해설: Undercut은 반대로 전류가 너무 높고 운봉 속도가 빠를 때, 모재가 과도하게 녹아 파인 곳에 쇳물이 채워지지 못해 생깁니다.

9. (중) Crater 균열을 방지하려면 용접을 끝낼 때 Torch 스위치를 갑자기 끄지 않고 Down-slope 기능을 사용해 전류를 천천히 줄여야 한다. ( )
정답: O
해설: 전류를 천천히 줄이면 용융지가 서서히 식으면서 수축에 의한 균열(Crater crack)을 막을 수 있습니다.

10. (상) Tungsten inclusion(텅스텐 혼입) 결함이 발생했을 경우, 결함 부위를 갈아내지 않고 그 위에 고전류로 덧살을 올려 용접하면 텅스텐이 알아서 녹아 없어진다. ( )
정답: X
해설: 텅스텐의 융점(약 3400도)은 철(약 1500도)보다 훨씬 높아 용접열로는 녹지 않습니다. 반드시 Grinder 등으로 완전히 파내고 재용접해야 합니다.

[카테고리 6: 비파괴 검사]
11. (중) MT(Magnetic Particle Testing)는 표면 결함뿐만 아니라 표면 바로 아래(표면 하)에 있는 미세한 내부 결함도 일부 검출할 수 있다. ( )
정답: O
해설: MT는 자력선의 누설을 이용하므로 표면에 열려 있지 않은 2~3mm 이내의 얕은 표면 하 결함도 찾아낼 수 있습니다.

[카테고리 7: 재료 및 성분]
12. (하) 연강(Mild steel)은 탄소 함량이 매우 적어 열을 가해도 쉽게 갈라지므로 용접성이 매우 나쁜 재료이다. ( )
정답: X
해설: 탄소량이 적은 연강(저탄소강)은 열영향부 경화나 저온 균열 위험이 적어 모든 강철 중에서 용접성이 가장 우수한 재료입니다.

13. (상) 오스테나이트계 스테인리스강(300계열)은 자석에 아주 잘 붙는 강자성체이므로 MT(자분 탐상) 검사가 가능하다. ( )
정답: X
해설: 오스테나이트계 STS는 니켈(Ni) 함유로 인해 자성을 띠지 않는 비자성체입니다. 따라서 MT는 불가능하며 PT(침투 탐상)를 사용해야 합니다.

[카테고리 8: 용접 극성]
14. (하) 알루미늄 용접 시 표면의 산화막을 깨트리는 현상을 청정 작용(Cleaning action)이라 하며, 이는 AC 전원의 DCEP(직류 역극성) 구간에서 발생한다. ( )
정답: O
해설: DCEP 구간에서 모재(-)로부터 튀어 나가는 전자가 두꺼운 산화알루미늄 피막을 깨는 역할을 합니다.

15. (중) DCEN(직류 정극성)은 모재가 (+), 전극이 (-)이므로 전자가 모재를 떠나 전극으로 충돌하기 때문에 전극봉이 쉽게 과열되어 녹아내린다. ( )
정답: X
해설: 설명이 반대로 되었습니다. DCEN은 전자가 전극(-)을 떠나 모재(+)에 충돌하므로 모재가 깊게 녹습니다. 전극봉이 녹아내리는 것은 DCEP(역극성)입니다.

[카테고리 9: 전원 특성]
16. (중) 수하 특성 전원은 Arc length가 짧아져 전압이 낮아질 때 전류가 미세하게 증가하여 Arc가 꺼지는 것을 막고 안정성을 유지해 준다. ( )
정답: O
해설: 전압과 전류가 반비례하는 곡선(수하 특성) 덕분에 단락(합선) 시에도 전류가 무한히 오르지 않고 안정적인 Arc를 유지합니다.

17. (하) 용접기의 무부하 전압(Open-circuit voltage)은 용접을 하지 않고 전원만 켜져 있을 때 양극단(Torch와 모재)에 걸려 있는 전압을 말한다. ( )
정답: O
해설: 기기 작동 전 대기 상태의 전압으로, Arc를 처음 발생시킬 때 필요한 힘입니다(전격 사고의 주원인이기도 합니다).

[카테고리 10: 입열량과 변형]
18. (하) 용접 입열량(Heat input)이 모재의 두께에 비해 너무 높으면 모재가 과열되어 팽창과 수축이 커지므로 변형(Deformation)이 심해진다. ( )
정답: O
해설: 입열량이 과다하면 열변형이 심해지고 결정립이 조대화되어 기계적 성질도 나빠집니다.

19. (중) 역변형법(Pre-setting)은 용접 후 수축으로 인해 모재가 휠 것을 예상하여, 사전에 미리 반대 방향으로 각도를 주어 가접하는 변형 방지법이다. ( )
정답: O
해설: 용접 변형을 역으로 이용하여 용접이 끝난 후 모재가 일직선으로 펴지게 만드는 가장 효과적인 조립 기술입니다.

20. (상) Peening(피닝) 작업은 초층(Root pass) 용접 직후에 실시해야 슬래그 박리성이 좋아지고 잔류 응력 제거 효과가 가장 크다. ( )
정답: X
해설: 초층 비드나 가장 표면 비드는 두께가 얇고 강도가 약해 Peening(망치질)을 하면 오히려 크랙(Crack)이 발생하기 쉬우므로 실시하지 않는 것이 원칙입니다.

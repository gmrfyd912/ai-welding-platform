=============================================================
GTAW 용접 이론 및 실습 평가 - OX 퀴즈형 (8차 세트)
=============================================================
* 난이도 비율: 하(50%), 중(30%), 상(20%)
* 카테고리: 안전, 장비/도구, 용접종류, 아크이해, 결함, 비파괴검사, 재료, 극성, 전원특성, 입열량/변형

[카테고리 1: 용접과 관련된 안전]
1. (하) GTAW는 SMAW처럼 눈에 보이는 시커먼 연기(Fume)가 많이 나지 않으므로, 환기 장치나 방진 마스크 없이 작업해도 호흡기 질환의 위험이 전혀 없다. ( )
정답: X
해설: GTAW에서도 눈에 잘 보이지 않는 미세한 금속 Fume과 오존(O3) 등 유해 가스가 발생하므로 환기와 마스크 착용은 필수입니다.

2. (하) Earth(접지) Clamp를 모재에 물릴 때는 페인트나 녹이 슬어 있는 부위를 피해서 전기가 잘 통하는 깨끗한 금속 표면에 단단히 고정해야 한다. ( )
정답: O
해설: 접지 부위의 전기 저항이 높으면 Arc가 불안정해지고 Arc blow 현상이나 단자 과열이 발생할 수 있습니다.

[카테고리 2: 장비 및 도구]
3. (하) Torch 스위치는 단순히 Arc를 켜고 끄는 기능만 하므로, 전류를 서서히 줄여주는 Crater fill 기능은 스위치 조작과 무관하다. ( )
정답: X
해설: 용접기 패널에서 Crater 기능을 켜두면, 스위치를 놓았을 때(Off) 아크가 바로 꺼지지 않고 설정된 Down-slope 시간에 맞춰 전류가 서서히 줄어듭니다.

4. (상) Gas lens를 장착하면 Shielding gas가 직진성이 강한 Laminar flow(층류)로 분사되므로, 코너 부위나 깊은 홈 용접 시 Tungsten 전극을 평소보다 더 길게 빼서 용접할 수 있다. ( )
정답: O
해설: 층류는 가스 보호 영역을 길고 안정적으로 유지시켜 주므로 전극봉의 돌출 길이(Stick-out)를 더 늘려 시야를 확보할 수 있게 해줍니다.

[카테고리 3: 용접의 종류]
5. (하) FCAW(Flux Cored Arc Welding)는 튜브 형태의 와이어 내부에 쇳물을 덮어 보호하는 Flux가 채워져 있는 소모성 전극 용접이다. ( )
정답: O
해설: 와이어 자체가 녹으면서 내부의 Flux가 슬래그를 형성해 쇳물을 보호하는 반자동 용접법입니다.

6. (중) Brazing(경납땜)은 접합하고자 하는 두 모재의 끝을 완전히 용융점 이상으로 녹여서 하나로 합치는 융접(Fusion welding)의 일종이다. ( )
정답: X
해설: Brazing은 모재를 녹이지 않고, 모재보다 용융점이 낮은 용가재만 녹여서 틈새로 스며들게 하는 모세관 현상을 이용한 접합법입니다.

[카테고리 4: 아크의 이해]
7. (하) Arc length가 짧아질수록 전기적 저항이 줄어들기 때문에 Arc 전압은 감소한다. ( )
정답: O
해설: Arc 길이는 전압과 정비례하는 성질을 가지고 있습니다.

8. (중) Arc blow(자기 쏠림) 현상은 전류의 방향이 계속해서 바뀌는 AC(교류) 용접을 할 때 가장 심하게 발생한다. ( )
정답: X
해설: AC는 극성이 초당 수십 번씩 바뀌어 자기장이 상쇄되므로 Arc blow가 거의 발생하지 않습니다. 주로 DC(직류) 용접에서 문제가 됩니다.

[카테고리 5: 용접 결함]
9. (하) Undercut 결함은 용접 열이 너무 강해 모재가 파인 후 그 공간을 쇳물(용가재)이 채워주지 못해서 발생한 골짜기 형태의 결함이다. ( )
정답: O
해설: 과전류나 너무 빠른 운봉으로 인해 발생하는 대표적인 형상 결함입니다.

10. (중) Tungsten inclusion(텅스텐 혼입) 결함이 발생한 쇳물 위에 용접 전류를 높여서 덧씌우기(Pass)를 하면 내부의 텅스텐 조각이 자연스럽게 녹아 없어진다. ( )
정답: X
해설: 텅스텐의 융점은 철보다 훨씬 높아 용접 열로는 녹지 않고 그대로 내부에 남아 응력 집중부가 되므로 반드시 그라인더로 파내야 합니다.

11. (상) Hot crack(고온 균열)은 용접부가 쇳물 상태에서 응고(Solidification)될 때, 황(S) 등의 불순물이 결정 입계에 남아 결합력을 약하게 만들어 갈라지는 현상이다. ( )
정답: O
해설: 금속이 굳는 고온 단계에서 수축 응력을 견디지 못하고 불순물이 모인 취약한 부위가 갈라지는 현상입니다.

[카테고리 6: 비파괴 검사]
12. (하) RT(방사선 투과 검사)를 실시할 때는 방사선 피폭 위험이 있으므로 반드시 작업자들을 대피시키고 통제 구역을 설정해야 한다. ( )
정답: O
해설: X선이나 감마선은 인체에 매우 치명적이므로 안전 보건 수칙에 따라 엄격한 격리가 필요합니다.

13. (중) MT(자분 탐상 검사)는 알루미늄이나 동(Copper) 합금 같은 비자성 금속의 미세한 표면 크랙을 찾는 데 매우 적합하다. ( )
정답: X
해설: MT는 자석에 붙는 강자성체 전용 검사법이므로 알루미늄이나 동 합금에는 절대 사용할 수 없고 PT를 사용해야 합니다.

[카테고리 7: 재료 및 성분]
14. (하) 가장 널리 쓰이는 철강 재료인 탄소강(Carbon steel)은 철(Fe)에 일정량의 탄소(C)가 합금되어 강도가 조절된 금속이다. ( )
정답: O
해설: 탄소 함유량에 따라 저탄소강, 중탄소강, 고탄소강으로 분류되며 탄소가 많을수록 단단해집니다.

15. (중) 304, 316 계열의 오스테나이트계 STS(스테인리스강)는 용접 후 자석을 대보면 찰칵하고 아주 강하게 달라붙는 특징이 있다. ( )
정답: X
해설: 오스테나이트계 STS는 니켈(Ni) 성분으로 인해 자성을 잃어버린 비자성체이므로 자석에 붙지 않습니다.

[카테고리 8: 용접 극성]
16. (하) DCEN(직류 정극성)은 전자가 모재에서 출발하여 Tungsten 전극으로 충돌하므로 전극봉이 쉽게 녹아내리는 극성이다. ( )
정답: X
해설: 설명이 반대입니다. DCEN은 전자가 전극(-)에서 출발하여 모재(+)로 충돌하므로 전극봉은 보호되고 모재가 깊게 녹습니다.

17. (상) 알루미늄 교류(AC) 용접에서 산화막을 파괴하는 Cleaning action(청정 작용)은 1사이클 중 모재가 (-), 전극이 (+)가 되는 EP(역극성) 구간에서 일어난다. ( )
정답: O
해설: 모재(-)에서 전극(+) 방향으로 튀어나가는 전자가 표면의 끈질긴 산화알루미늄 피막을 깨부수게 됩니다.

[카테고리 9: 전원 특성]
18. (하) 용접기의 전원 특성 중 Constant Current(정전류 특성)는 아크 길이가 길어지든 짧아지든 전압이 단 1볼트(V)도 변하지 않는 성질을 말한다. ( )
정답: X
해설: 정전류 특성은 전압이 변할 때 전류(Current) 값이 거의 변하지 않도록 설계된 것이지 전압이 고정된 것이 아닙니다.

19. (상) 인버터(Inverter)형 용접기는 입력된 교류를 직류로 바꾼 뒤 다시 수십 kHz의 고주파 교류로 변환하고 다시 직류로 정류하는 과정을 거쳐 변압기 크기를 획기적으로 줄인 방식이다. ( )
정답: O
해설: 주파수가 높아질수록 동일한 전력을 변환하는 트랜스포머(변압기)의 코어 크기를 대폭 소형화, 경량화할 수 있는 인버터 회로의 원리입니다.

[카테고리 10: 입열량과 변형]
20. (중) Weaving(위빙) 폭을 너무 넓게 가져가면 용접의 진행 속도(운봉 속도)가 느려지므로 결과적으로 모재 단위 길이당 Heat input(입열량)은 감소하게 된다. ( )
정답: X
해설: 운봉 속도가 느려지면 같은 구간에 열이 더 오래 머물게 되므로 Heat input(입열량)은 대폭 증가하게 되어 열변형과 결정립 조대화의 원인이 됩니다.

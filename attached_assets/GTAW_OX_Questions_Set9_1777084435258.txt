=============================================================
GTAW 용접 이론 및 실습 평가 - OX 퀴즈형 (9차 세트)
=============================================================
* 난이도 비율: 하(50%), 중(30%), 상(20%)
* 카테고리: 안전, 장비/도구, 용접종류, 아크이해, 결함, 비파괴검사, 재료, 극성, 전원특성, 입열량/변형

[카테고리 1: 용접과 관련된 안전]
1. (하) 밀폐된 공간에서 Ar gas 누출 시, 공기보다 가벼워 위로 날아가므로 바닥에 누워있으면 질식을 피할 수 있다. ( )
정답: X
해설: Ar gas는 공기보다 무거워 바닥부터 쌓이므로, 밀폐 공간의 하단부에 있으면 산소 결핍으로 인한 질식 위험이 가장 큽니다.

2. (중) GTAW Torch를 사용할 때 발생하는 High frequency는 주변의 Pacemaker를 착용한 사람에게 치명적인 영향을 줄 수 있다. ( )
정답: O
해설: 고주파는 강력한 전자기파 간섭을 일으켜 심박 조율기 같은 정밀 의료 기기의 오작동을 유발할 수 있으므로 접근을 엄격히 금지해야 합니다.

[카테고리 2: 장비 및 도구]
3. (하) Gas lens를 Torch에 장착하면 가스의 난류를 Laminar flow로 바꿔주므로, 좁은 Groove 용접 시 Tungsten 전극을 평소보다 더 길게 뽑아 쓸 수 있다. ( )
정답: O
해설: Gas lens는 보호 가스의 직진성을 높여 보호 범위가 늘어나므로 전극봉 돌출 길이를 연장할 수 있습니다.

4. (상) Water-cooled Torch 시스템에서 냉각수가 흐르지 않을 때 용접기 출력을 차단하는 안전 장치를 Flow switch라고 한다. ( )
정답: O
해설: Flow switch는 냉각수의 유량을 감지하여 물이 순환하지 않으면 즉시 전원을 끊어 Power cable이 타버리는 것을 막습니다.

[카테고리 3: 용접의 종류]
5. (하) FCAW는 비소모성 전극을 사용하며 별도의 용가재를 손으로 공급하는 수동 용접이다. ( )
정답: X
해설: FCAW는 내부가 Flux로 채워진 와이어가 자동으로 연속 공급되며 녹는 소모성 전극 방식의 반자동 용접입니다.

6. (중) Friction welding은 전기 Arc의 열을 이용하여 두 모재의 끝단을 녹인 후 압력을 가해 붙이는 용융 용접법이다. ( )
정답: X
해설: Friction welding은 외부 열원 없이 모재를 고속으로 회전시킬 때 발생하는 기계적 마찰열로 금속을 부드럽게 한 후 압착하는 Solid-state welding입니다.

[카테고리 4: 아크의 이해]
7. (하) 용접 Current를 높게 설정할수록 Arc 열에너지가 강해져서 모재의 Penetration 깊이가 깊어진다. ( )
정답: O
해설: 용접 전류는 입열량과 직결되므로 전류가 증가하면 금속을 녹이는 힘도 강해집니다.

8. (중) Arc blow 현상을 줄이기 위해서는 Earth를 용접부에서 최대한 멀리 떨어진 곳에 한 군데만 연결해야 한다. ( )
정답: X
해설: Earth가 멀거나 한쪽으로 치우치면 자기장의 불균형이 심해져 Arc blow가 악화됩니다. 최대한 가까이, 혹은 양쪽으로 분산시켜야 합니다.

[카테고리 5: 용접 결함]
9. (하) Undercut 결함은 용접 전류가 너무 과도하거나 운봉 속도가 너무 빠를 때, 모재가 파인 부분을 쇳물이 채워주지 못해서 발생한다. ( )
정답: O
해설: Undercut은 지나친 열로 모재가 녹아내려 형성된 골짜기 형태의 치명적인 결함입니다.

10. (중) Porosity는 용접 쇳물 속에 녹아있던 Gas가 응고 과정에서 밖으로 빠져나오지 못하고 내부에 갇혀 생성된 구멍이다. ( )
정답: O
해설: 표면 오염물이나 Shielding gas 부족으로 인해 침투한 가스가 응고 시 배출되지 못해 생기는 내부 결함입니다.

11. (상) Cold crack은 용접부가 응고되는 뜨거운 상태에서 발생하며, 주로 황(S)이나 인(P) 같은 불순물이 원인이다. ( )
정답: X
해설: 뜨거운 상태에서 불순물에 의해 갈라지는 것은 Hot crack입니다. Cold crack은 냉각 후 수소 가스에 의해 시간차를 두고 발생합니다.

[카테고리 6: 비파괴 검사]
12. (하) VT는 육안과 돋보기, 용접 게이지 등을 사용하여 Bead 표면의 결함이나 치수를 확인하는 가장 기본적이고 선행되어야 할 비파괴 검사이다. ( )
정답: O
해설: 모든 비파괴 검사의 첫 단계는 VT이며, 표면이 합격해야 내부 검사를 진행할 수 있습니다.

13. (중) UT는 결함의 종류를 필름으로 직관적으로 보여주므로 RT보다 결함 형태 판독이 훨씬 쉽다. ( )
정답: X
해설: 직관적인 형태 판독은 RT가 뛰어나며, UT는 초음파 파형을 보므로 결함의 깊이와 위치 측정에 유리하지만 형태 판독은 어렵습니다.

[카테고리 7: 재료 및 성분]
14. (하) 오스테나이트계 STS 용접 시 Chrome fume이 발생하므로, 반드시 방진마스크를 착용하고 국소 배기 장치를 가동해야 한다. ( )
정답: O
해설: 크롬이 고온에서 타면서 발생하는 6가 크롬 흄은 발암 물질이므로 철저한 호흡기 보호가 필수입니다.

15. (상) 탄소강 용접 시 Pre-heating 온도를 너무 높게 설정하여 오랫동안 유지하면, 오히려 결정립이 조대화되어 충격에 약해질 수 있다. ( )
정답: O
해설: 적절한 예열은 좋으나, 과도한 예열과 장시간 유지는 HAZ의 금속 조직을 거칠게 만들어 인성 저하를 초래합니다.

[카테고리 8: 용접 극성]
16. (하) DCEN은 전자가 모재에서 출발하여 Tungsten 전극에 충돌하는 극성이다. ( )
정답: X
해설: DCEN은 전자가 전극에서 출발하여 모재로 충돌하므로 모재 쪽에 열이 약 70% 집중되어 깊은 Penetration을 만듭니다.

17. (중) AC를 사용하는 Aluminum 용접에서 AC Balance 설정을 EP 비율 90%로 극단적으로 높이면, 산화막은 잘 깨지지만 Tungsten 전극이 녹아 떨어질 위험이 매우 커진다. ( )
정답: O
해설: EP 구간은 전극에 열이 집중되므로, 비율이 너무 높으면 전극이 견디지 못하고 녹아 쇳물로 떨어지게 됩니다.

[카테고리 9: 전원 특성]
18. (하) Drooping characteristic(수하 특성)을 가진 수동 용접기는 작업자가 손을 떨어 Arc length가 변해도 Current 값이 거의 일정하게 유지된다. ( )
정답: O
해설: 수하 특성 덕분에 전압 변동에 따른 전류 변화폭이 적어 수동 용접 시 안정적인 Bead를 만들 수 있습니다.

19. (상) Constant Voltage(정전압 특성) 전원에서는 Torch가 모재에서 멀어져 Arc length가 길어지면, 기계가 자체적으로 전류를 급격히 낮춰 와이어 송급 속도와 균형을 맞추는 Self-regulating이 작동한다. ( )
정답: O
해설: 이것이 반자동 용접에서 정전압 특성을 사용하는 핵심 이유이며, 일정한 Arc 길이를 스스로 복원합니다.

[카테고리 10: 입열량과 변형]
20. (중) 용접 진행 방향과 반대로 일정 구간씩 짧게 나누어 Back-step으로 용접하면, 열이 한쪽으로 누적되지 않아 모재의 Deformation을 효과적으로 줄일 수 있다. ( )
정답: O
해설: Back-step법은 잔류 응력을 상쇄시키고 열 분포를 균일하게 하여 변형을 억제하는 대표적인 운봉 기법입니다.

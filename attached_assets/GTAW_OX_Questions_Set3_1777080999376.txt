=============================================================
GTAW(TIG) 용접 이론 및 실습 평가 - OX 퀴즈형 (3차 세트)
=============================================================
* 난이도 비율: 하(50%), 중(30%), 상(20%)
* 카테고리: 안전, 장비/도구, 용접종류, 아크이해, 결함, 비파괴검사, 재료, 극성, 전원특성, 입열량/변형

[카테고리 1: 용접과 관련된 안전]
1. (하) 가스 실린더(봄베)는 전도 방지를 위해 체인 등으로 고정해야 하며, 직사광선이 내리쬐는 곳에 보관해야 한다. ( )
정답: X
해설: 가스 실린더는 폭발 위험이 있으므로 직사광선을 피해 40도 이하의 서늘하고 통풍이 잘되는 곳에 보관해야 합니다.

2. (상) 용접 중 눈에 이물질(슬래그나 불꽃)이 들어갔을 때는 즉시 손으로 비벼서 제거해야 한다. ( )
정답: X
해설: 눈을 비비면 각막에 상처가 날 수 있으므로 흐르는 물이나 식염수로 씻어내거나 병원을 방문해야 합니다.

[카테고리 2: 장비 및 도구]
3. (하) GTAW용 텅스텐 전극봉을 연마할 때는 연마 결이 전극봉의 길이 방향(세로)이 되도록 해야 아크가 안정된다. ( )
정답: O
해설: 가로 방향으로 연마하면 아크가 흔들리거나 불안정해질 수 있습니다.

4. (중) 가스 쉴드 유량이 너무 적으면 대기가 유입되어 산화 결함이 생기지만, 너무 많아도 난류에 의해 결함이 생길 수 있다. ( )
정답: O
해설: 적정 유량 유지가 가장 중요합니다.

[카테고리 3: 용접의 종류]
5. (하) 솔더링(연납땜)은 모재를 용융시키지 않고 용가재만 녹여 접합하는 방식이다. ( )
정답: O
해설: 납땜의 정의는 모재의 용융점보다 낮은 용가재를 사용하는 비용융 접합입니다.

6. (중) 테르밋 용접(Thermit welding)은 전기에너지를 사용하여 금속 산화물과 알루미늄 가루를 반응시킨다. ( )
정답: X
해설: 테르밋 용접은 외부 전기가 아니라 화학 반응(산화-환원) 시 발생하는 강한 열을 이용합니다.

[카테고리 4: 아크의 이해]
7. (하) 아크 용접 시 아크 전압은 아크 길이에 비례하여 변한다. ( )
정답: O
해설: 아크가 길어지면 저항이 커져서 전압이 올라가고, 짧아지면 전압이 내려갑니다.

8. (중) 용접 아크의 온도는 중심부에서 약 5,000 ~ 20,000℃에 달하는 고온의 상태이다. ( )
정답: O
해설: 아크는 플라즈마 상태로 매우 높은 열에너지를 가집니다.

[카테고리 5: 용접 결함]
9. (하) 스패터(Spatter)는 용접 중 튀어나오는 금속 알갱이로, GTAW 용접에서는 SMAW(피복아크)에 비해 거의 발생하지 않는다. ( )
정답: O
해설: GTAW는 아크가 매우 깨끗하고 정적이며 불순물이 적어 스패터가 거의 없습니다.

10. (중) 언더필(Underfill) 결함은 비드 표면이 모재보다 높게 솟아오른 상태를 말한다. ( )
정답: X
해설: 언더필은 비드가 모재 표면보다 낮은(충전 부족) 상태를 말합니다.

11. (상) 용접부의 '웜홀(Worm hole)'은 기공이 마치 벌레가 지나간 자국처럼 길게 이어진 형태의 결함이다. ( )
정답: O
해설: 응고 속도와 가스 배출 속도의 차이로 인해 관 모양의 기공이 생기는 현상입니다.

[카테고리 6: 비파괴 검사]
12. (하) 자분탐상검사(MT)는 비자성체인 구리나 알루미늄 용접부 검사에도 널리 사용된다. ( )
정답: X
해설: 자분탐상은 자석에 붙는 강자성체(철 등)에만 사용 가능합니다.

13. (중) 방사선 투과 시험(RT) 결과 필름에서 기공(Porosity)은 주변보다 검은색 점으로 나타난다. ( )
정답: O
해설: 기공 부분은 금속이 없어 방사선이 더 많이 투과되므로 필름이 더 검게 노출됩니다.

[카테고리 7: 재료 및 성분]
14. (하) 연강(저탄소강)은 탄소 함유량이 적어 용접성이 매우 우수하며 일반적인 구조물에 많이 쓰인다. ( )
정답: O
해설: 탄소량이 0.3% 이하인 연강은 균열 위험이 적어 용접이 쉽습니다.

15. (상) 스테인리스강 용접 시 발생하는 크롬 6가 흄은 인체에 무해하므로 별도의 마스크가 필요 없다. ( )
정답: X
해설: 크롬 흄은 발암 물질로 분류될 만큼 매우 유해하므로 전용 방진마스크와 환기가 필수입니다.

[카테고리 8: 용접 극성]
16. (하) GTAW에서 직류 역극성(DCEP)은 모재의 용입이 가장 깊게 나오는 극성이다. ( )
정답: X
해설: 역극성은 용입이 가장 얕고 전극봉이 쉽게 녹습니다. 용입이 깊은 것은 정극성(DCEN)입니다.

17. (중) 알루미늄 용접 시 교류(AC)를 사용하는 이유는 반 사이클마다 전극에서 전자가 튀어나와 표면의 산화막을 파괴하기 위해서이다. ( )
정답: O
해설: 이것이 교류의 청정 작용(Cleaning Action)입니다.

[카테고리 9: 전원 특성]
18. (하) 용접기의 정전류 특성은 아크 길이가 변해도 용접 전류값이 일정하게 유지되려는 성질이다. ( )
정답: O
해설: 수동 용접인 TIG와 피복아크용접기의 핵심 특성입니다.

[카테고리 10: 입열량과 변형]
19. (하) 용접 입열량이 과도하면 결정립이 조대화되어 용접부의 인성이 떨어진다. ( )
정답: O
해설: 너무 많은 열을 주면 금속 조직이 커져서 충격에 약해집니다.

20. (중) 용접 변형을 방지하기 위해 가접(Tack weld)을 튼튼하게 하고 순서를 대칭적으로 가져가는 것이 좋다. ( )
정답: O
해설: 열의 균형을 맞추고 구조적 강성을 높여 변형을 억제하는 기본 원칙입니다.

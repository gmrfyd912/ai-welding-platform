=============================================================
GTAW 용접 이론 및 실습 평가 - OX 퀴즈형 (4차 세트)
=============================================================
* 난이도 비율: 하(50%), 중(30%), 상(20%)
* 카테고리: 안전, 장비/도구, 용접종류, 아크이해, 결함, 비파괴검사, 재료, 극성, 전원특성, 입열량/변형

[카테고리 1: 용접과 관련된 안전]
1. (하) 용접용 가죽장갑은 화상 예방뿐만 아니라 감전을 막아주는 절연체 역할도 하므로 항상 건조한 상태를 유지해야 한다. ( )
정답: O
해설: 땀이나 물에 젖은 장갑은 전기 저항이 낮아져 감전 위험이 매우 커집니다.

2. (하) Ar(아르곤) 가스는 공기보다 가벼우므로, 가스 누출 시 천장 쪽에 모이게 된다. ( )
정답: X
해설: Ar 가스는 공기보다 무거워 바닥에 가라앉으므로 밀폐공간 하단의 환기가 중요합니다.

[카테고리 2: 장비 및 도구]
3. (중) Gas lens를 사용하면 난류가 생겨 가스 보호막이 쉽게 흩어지므로 야외 용접에 적합하다. ( )
정답: X
해설: Gas lens는 가스를 층류(Laminar flow)로 곧게 뻗게 만들어 보호 효과를 높여주는 장치입니다.

4. (상) Pulse TIG에서 Background current(베이스 전류)는 Arc가 꺼지지 않게 유지하면서 용융지를 순간적으로 냉각시키는 역할을 한다. ( )
정답: O
해설: Peak current로 녹이고 Background current로 식히는 과정을 반복하여 열 조절을 쉽게 합니다.

[카테고리 3: 용접의 종류]
5. (하) SMAW(피복아크용접)는 용접봉 자체가 녹아서 쇳물이 되는 소모성 전극 용접이다. ( )
정답: O
해설: SMAW는 전극(용접봉)이 닳아 없어지지만, GTAW는 Tungsten 전극이 소모되지 않습니다.

6. (중) 마찰용접(Friction welding)은 고상용접(Solid-state welding)의 일종으로, 외부에서 가스 토치로 열을 가한 뒤 압력을 주어 접합한다. ( )
정답: X
해설: 외부 열원이 아니라 두 부품을 직접 회전시켜 발생하는 기계적 마찰열로 결합하는 방식입니다.

[카테고리 4: 아크의 이해]
7. (하) DCEN(직류 정극성)에서는 Arc 열의 약 70%가 모재에 집중된다. ( )
정답: O
해설: 전자가 전극(-)에서 튀어나와 모재(+)에 충돌하면서 모재 쪽에 더 큰 열을 발생시킵니다.

8. (하) GTAW 용접 시 Arc length가 너무 길면 가스 보호 범위를 벗어나 Porosity(기공)가 발생할 확률이 높다. ( )
정답: O
해설: Arc가 길면 Shielding gas 커버리지가 분산되어 외부 공기가 침투하기 쉽습니다.

[카테고리 5: 용접 결함]
9. (하) Undercut 결함은 용접 전류가 너무 낮아서 모재가 충분히 녹지 않았을 때 주로 발생한다. ( )
정답: X
해설: Undercut은 전류가 너무 높거나 운봉이 너무 빨라 모재가 과도하게 파인 후 쇳물이 채워지지 않은 결함입니다.

10. (중) Overlap 결함은 용융 금속이 모재 표면과 융합(Fusion)되지 않고 단순히 덮어 씌워진 상태를 말한다. ( )
정답: O
해설: 쇳물이 너무 차갑거나 운봉이 느려 모재를 녹이지 못한 채 얹혀진 불량입니다.

11. (상) 저온 균열(Cold crack)은 용접이 끝난 후 금속이 식으면서 주로 수소(Hydrogen) 가스에 의해 유발된다. ( )
정답: O
해설: 수소 지연 균열이라고도 하며, 용접부에 침투한 수소가 굳은 후 팽창하면서 균열을 일으킵니다.

[카테고리 6: 비파괴 검사]
12. (하) RT(방사선 투과 검사)는 X-ray나 감마선을 쏘아 필름에 맺힌 상을 보고 내부 결함을 판단한다. ( )
정답: O
해설: 내부 기공이나 불순물을 직관적인 사진으로 확인할 수 있는 대표적인 내부 검사법입니다.

13. (중) MT(자분 탐상 검사)는 알루미늄이나 오스테나이트계 스테인리스강의 표면 결함 검사에 매우 효과적이다. ( )
정답: X
해설: MT는 자석에 붙는 강자성체(일반 철재)에만 사용할 수 있습니다.

[카테고리 7: 재료 및 성분]
14. (하) 스테인리스강(STS)은 크롬(Cr) 성분이 표면에 치밀한 산화막을 형성하여 녹이 스는 것을 방지한다. ( )
정답: O
해설: 크롬이 11~12% 이상 함유되어 부동태 피막을 형성하는 것이 스테인리스의 핵심입니다.

15. (상) 두꺼운 알루미늄 판을 용접할 때는 열전도율이 매우 높아 예열(Pre-heating)을 하는 것이 절대 금지되어 있다. ( )
정답: X
해설: 알루미늄은 열전도율이 너무 높아 열을 금방 빼앗기므로 두꺼운 모재는 초기 용입을 얻기 위해 예열을 실시합니다.

[카테고리 8: 용접 극성]
16. (하) 알루미늄 용접에 교류(AC)를 사용하는 주된 이유는 전극의 플러스 구간에서 산화막을 파괴하는 청정 작용을 얻기 위해서이다. ( )
정답: O
해설: 교류의 DCEP(역극성) 사이클이 산화알루미늄 피막을 깨주는 필수적인 역할을 합니다.

[카테고리 9: 전원 특성]
17. (중) 수동 GTAW 용접기는 아크 길이가 변해도 전압이 일정하게 유지되는 '정전압 특성'을 사용한다. ( )
정답: X
해설: 작업자의 손떨림에 대비해 전류가 일정하게 유지되는 '정전류 특성(수하 특성)'을 사용합니다. 정전압은 반자동(GMAW)용입니다.

[카테고리 10: 입열량과 변형]
18. (하) 용접 속도(V)를 빠르게 하면 용접부 단위 길이당 들어가는 입열량(Heat input)은 증가한다. ( )
정답: X
해설: 입열량은 속도에 반비례하므로 속도를 빠르게 하면 열이 적게 들어가 입열량이 감소합니다.

19. (중) 용접 부위를 해머 등으로 두드려주는 Peening 작업은 용접부의 잔류 응력을 완화시키는 효과가 있다. ( )
정답: O
해설: 표면을 타격하여 소성 변형을 주면 수축하려는 응력을 풀어주어 균열을 방지합니다.

20. (상) 구속 지그(Jig)로 모재를 강하게 고정해 놓고 용접하면 눈에 보이는 외부 변형은 줄어들지만 내부에 갇힌 잔류 응력은 커진다. ( )
정답: O
해설: 외형적 틀어짐(각변형)은 억제할 수 있으나, 수축하려는 힘이 내부에 쌓여 균열(Crack)의 위험성이 높아집니다.
